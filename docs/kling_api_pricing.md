以下接口为透明代理，**不涉及计费**：

- ❌ GET `/v1/general/presets-voices` - 查询预设声音列表
- ❌ DELETE `/v1/general/delete-voices` - 删除声音

## 📊 计费模式总结

### 同步接口（立即扣费）
1. ✅ **identify-face** (人脸识别) - 0.05元/次
2. ✅ **custom-elements** (自定义元素) - 费用待确认
3. ❓ **image-recognize** (图像识别) - 0.1元/次（需测试确认）
4. ❓ **tts** (语音合成) - 0.05元/次（需测试确认）

### 异步接口 - 固定费用（任务成功后扣费）
1. ✅ **custom-voices** (音色定制) - 0.05元/次
2. ⚠️ **text-to-audio** (文生音效) - 0.25元/次（需测试确认）
3. ⚠️ **video-to-audio** (视频配音效) - 0.25元/次（需测试确认）

### 异步接口 - 按时长计费（任务成功后按实际时长扣费）
1. **视频生成类** - 按实际生成时长计费
2. **数字人** - 按秒计费
3. **对口型** - 按时长计费
4. **动作控制** - 按秒计费

### 异步接口 - 按张计费（任务成功后按张数扣费）
1. **图片生成类** - 按实际生成张数计费

### 不计费接口
- 所有查询和管理接口（GET/DELETE）

---

## ⚠️ 重要提醒

1. **视频时长计费**: Video-O1 模型按秒计费，其他模型按档位（5s/10s）计费
2. **图片按张计费**: 每生成一张图片扣减相应积分
3. **同步接口**: 立即返回结果并扣费，无需回调
4. **异步接口**: 提交任务后返回 task_id，需要通过回调或查询获取结果，成功后扣费
5. **回调扣费**: 异步接口在收到成功回调后，根据实际生成的时长/张数重新计算并扣费
